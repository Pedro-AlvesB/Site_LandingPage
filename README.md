# Projeto de Conclusão de Semestre: Landing Page

## Apresentação

Este repositório contém o projeto final de semestre das disciplinas de Web Standards e Projeto Integrador, com foco em HTML e CSS. O objetivo foi criar uma landing page responsiva e funcional que demonstra o domínio de conceitos de estrutura semântica, layout e estilização visual.

## Descrição do projeto

- Página única (one-page) com seções de hero, benefícios, sobre, contato e rodapé
- Layout responsivo e compatível com dispositivos móveis
- Uso de CSS moderno e práticas de design acessível

## Estrutura de arquivos

- `index.html`
- `styles.css`

## Como usar

1. Abra `index.html` no navegador.
2. Personalize o conteúdo do HTML conforme seu produto/serviço.
3. Ajuste estilos em `styles.css` conforme identidade visual desejada.

## Conclusão

Este projeto consolida o aprendizado prático do semestre em um produto tangível para portfólio. A landing page serve como demonstração de capacidade de implementação web front-end, organização de código e atenção à experiência do usuário.
